Мощные функции восстановления

Наша утилита-антидот содержит всё необходимое для восстановления вашей системы Windows до полной функциональности после атаки вредоносного ПО.

Защита системы

Восстанавливает Windows Defender, настройки брандмауэра и другие функции безопасности, отключенные вредоносным ПО.

Восстановление инструментов

Повторно включает Диспетчер задач, Редактор реестра, Командную строку и другие системные инструменты.

Сброс сети

Сбрасывает конфигурацию сети, очищает кэш DNS и восстанавливает нормальное подключение к интернету.

Восстановление реестра

Исправляет изменения реестра, сделанные вредоносным ПО для блокировки системных функций и функций безопасности.

Восстановление загрузочного сектора

Восстанавливает поврежденные загрузочные секторы для обеспечения нормального запуска системы после удаления вредоносного ПО.

Расширенное восстановление

Восстанавливает доступ к панели управления, ассоциациям файлов и удаляет автозагрузку вредоносного ПО.

Как это работает

Наш антидот работает путем систематического устранения ограничений и повреждений, вызванных вредоносным ПО, восстанавливая нормальную функциональность вашей системы.

1

Запуск с правами администратора

Запустите антидот-скрипт с правами администратора, чтобы разрешить изменения на системном уровне.

2

Автоматическое обнаружение

Утилита автоматически обнаруживает системные ограничения и отключенные функции.

3

Восстановление системы

Методично восстанавливает доступ к системным инструментам, функциям безопасности и сетевым функциям.

4

Блокировка вредоносного ПО

Удаляет записи автозагрузки и завершает процессы, блокирующие функциональность системы.

5

Подробный отчет

Создает детальный отчет о всех предпринятых действиях и рекомендует следующие шаги.

Почему стоит выбрать наш антидот

Когда вредоносное ПО блокирует доступ к вашей системе, вам нужно надежное решение, чтобы вернуть контроль.

Как использовать антидот

Следуйте этим пошаговым инструкциям для восстановления вашей системы после атаки вредоносного ПО

Стандартный процесс восстановления

  1. Скачайте ANTIDOTE.exe или ANTIDOTE_UI.exe из Гитхаба - Сохраните программу на рабочий стол или в любое доступное место
  2. Запустите от имени администратора - Щелкните правой кнопкой мыши по скрипту и выберите "Запуск от имени администратора" (программа автоматически запросит права администратора)
  3. Дождитесь выполнения модулей восстановления
  4. Проверьте результаты - Отчет о восстановлении будет сохранен на рабочем столе (recovery_report.txt)
  5. Перезагрузите компьютер - Для полного применения всех изменений (некоторые функции требуют перезагрузки)

Совет

Программа автоматически создаст свои резервные копии в нескольких местах (Рабочий стол, Документы, временная папка). Рекомендуется также сохранить копию на USB-накопителе, если ваша система сильно ограничена. Для более удобного интерфейса используйте версию ANTIDOT_UI.exe.

Расширенные функции WinAntidote

Версии скрипта

WinAntidote доступен в двух версиях:

  • ANTIDOT - Консольная версия для работы в терминале
  • ANTIDOT_UI - Версия с графическим интерфейсом для удобного использования

Модули восстановления

WinAntidote включает следующие ключевые модули восстановления:

Системные инструменты
  • Восстановление Диспетчера задач
  • Восстановление Редактора реестра
  • Восстановление CMD и PowerShell
  • Восстановление Панели управления
Сеть и безопасность
  • Сброс сетевой конфигурации
  • Восстановление антивируса
  • Восстановление брандмауэра
  • Сброс настроек прокси
Интерфейс Windows
  • Восстановление рабочего стола
  • Восстановление системного трея
  • Восстановление кнопки Пуск
  • Восстановление контекстного меню
Служебные функции
  • Очистка временных файлов
  • Восстановление загрузочного сектора
  • Сброс групповых политик
  • Создание отчета о восстановлении
  • и другие!

Автоматическое получение прав администратора

Программа имеет несколько методов для получения прав администратора:

  1. Стандартный метод через ShellExecuteW
  2. Альтернативный метод через PowerShell (если стандартный метод не сработал)
  3. Работа с ограниченными функциями (если права не получены)

Методы экстренного восстановления

Эти методы предназначены для экстренных ситуаций, когда система значительно ограничена вредоносным ПО или блокировщиками. Используйте их, если обычный метод запуска недоступен.

Использование в безопасном режиме

  1. Перезагрузите компьютер в безопасном режиме (F8 при загрузке)
  2. В безопасном режиме многие блокировки могут не работать
  3. Запустите программу ANTIDOT.exe или ANTIDOT_UI.exe от имени администратора
  4. Дождитесь завершения процесса восстановления
  5. Перезагрузите компьютер в обычном режиме

Борьба с блокирующими скриптами

WinAntidote имеет функцию kill_blocking_scripts, которая:

  • Обнаруживает и завершает процессы блокировки по ключевым словам
  • Удаляет файлы-флаги блокировки из разных мест системы
  • Находит и удаляет скрытые копии скриптов блокировки в системных папках
  • Автоматически создает резервные копии самого себя в разных местах

Метод запуска с USB-накопителя

  1. Скопируйте ANTIDOT.exe или ANTIDOT_UI.exe на USB-накопитель
  2. Создайте на USB-накопителе файл run.bat со следующим содержимым:
    3. @echo off
    cd /d %~dp0
    python ANTIDOT.exe
    pause
  3. Вставьте USB-накопитель в заблокированный компьютер
  4. Запустите run.bat (если возможно) или используйте командную строку

Автоматическое обнаружение проблем

WinAntidote имеет следующие методы обнаружения проблем:

  • restore_desktop_via_screenshot - анализирует скриншот для проверки пустого рабочего стола
  • check_filesystem_integrity - проверяет целостность файловой системы
  • restore_audio - обнаруживает и восстанавливает звуковые функции
  • bypass_input_blocks - преодолевает блокировки клавиатуры и мыши
  • restore_hidden_windows - восстанавливает скрытые системные окна

Часто задаваемые вопросы

Получите ответы на самые распространенные вопросы об использовании нашей утилиты-антидота

Безопасно ли использовать антидот?

Да, наш антидот разработан только для восстановления функций Windows, которые были отключены вредоносным ПО. Он не удаляет пользовательские файлы и не вносит постоянных системных изменений, кроме восстановления нормальной функциональности.

Удаляет ли это вредоносное ПО, вызвавшее проблему?

Антидот фокусируется на восстановлении функциональности системы, а не на удалении вредоносного ПО. После восстановления работы системных инструментов вам следует запустить полное сканирование антивирусом, используя Windows Defender или другое решение безопасности.

Почему скрипт требует прав администратора?

Права администратора необходимы для внесения изменений в системные настройки, записи реестра и службы, которые были изменены вредоносным ПО. Без этих привилегий антидот не сможет восстановить большинство системных функций.

Будет ли это работать, если мой Диспетчер задач и Редактор реестра отключены?

Да! Антидот специально разработан для работы даже при отключении этих инструментов. Он может восстановить доступ к Диспетчеру задач, Редактору реестра, Командной строке и другим заблокированным системным инструментам.

Нужно ли перезагружать компьютер после запуска антидота?

Хотя многие функции восстанавливаются немедленно, рекомендуется перезагрузка системы для полного применения всех изменений, особенно связанных с системными службами и функциями безопасности.

Могу ли я использовать это на Windows 7/8/10/11?

Да, антидот совместим со всеми современными версиями Windows, включая Windows 7, 8, 10 и 11. Основные системные функции, которые он восстанавливает, существуют во всех этих версиях.

Что делать, если антидот не решает мою проблему?

Антидот создает подробный отчет о предпринятых действиях. Если некоторые функции не восстановлены, проверьте этот отчет на наличие подсказок. Вы также можете попробовать запустить в безопасном режиме или связаться с нами для дополнительной поддержки.

Это проект с открытым исходным кодом? Могу ли я его модифицировать?

Да, антидот имеет открытый исходный код, и вы можете модифицировать его под свои конкретные нужды. Мы поощряем пользователей вносить улучшения или адаптации для конкретных вариантов вредоносного ПО.
